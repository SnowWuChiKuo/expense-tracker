# 介紹
***
可以創建帳號、使用facebook登入，開始使用專屬自己的記帳程式。

# 功能
*** 
- 首頁可以瀏覽所有支出項目以及總支出
- 可以分類顯示不同類別的支出，總支出會依不同種類變化
- 可以新增新的支出
- 可以編輯單筆支出的詳細資訊
- 可以刪除你想刪除的支出

# 開始使用
***
1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟後，透過中顛機進入資料夾，輸入:
  npm install
4. 安裝完畢後，繼續輸入:
  npm run dev
5. 若看見以下文字代表順利運行，打開瀏覽器進入到以下網址
  App is running in http://localhost:3000
6. 新增種子資料
  npm run seed
